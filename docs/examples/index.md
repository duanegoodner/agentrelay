# Examples

These YAML files are validated in tests so they stay in sync with the current
`TaskGraphBuilder` schema.

- [minimal.yaml](minimal.yaml): minimal linear graph.
- [workstreams.yaml](workstreams.yaml): parent/child workstreams with explicit mapping.
- [mixed-default-and-explicit-workstreams.yaml](mixed-default-and-explicit-workstreams.yaml): explicit `default` plus additional workstream.
